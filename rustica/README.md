## Getting Started

### Prerequisites

* node.js
* npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lopezrunco/MaderaRustica-Web-app-React.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start app
   ```sh
   npm start
   ```
4. Build app
    ```sh
    npm run build
    ```
5. Compile & deploy to firebase
    ```sh
    npm run deploy
    ```
